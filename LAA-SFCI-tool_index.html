LAA-SFCI Calculator: A TEE-Based Left Atrial Appendage Structure–Function Coupling Tool

Enter LAA-VI and LAA functional parameters to calculate candidate LAA structure–function coupling indices. The tool provides an auxiliary AF disease-stage profile based on the study-derived parameter pattern. It is intended for research demonstration and should not be used as a stand-alone diagnostic or treatment decision tool.

Input Parameters

LAA-VI entered separately
Note: LAA-VI is required. LAA-GLS and LAA-SLE are recommended as priority inputs. Enter velocities in cm/s, LAA-SLE in mm, and LAA-FAC as a percentage (a decimal such as 0.30 is also accepted). No personally identifiable information should be entered.
